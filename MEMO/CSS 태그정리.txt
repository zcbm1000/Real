📌 자주 쓰이는 HTML 태그 정리
<html>문서 전체를 감싸는 루트 요소

HTML 문서임을 정의하기 위해 사용




<head> 문서의 메타데이터 영역

제목, CSS, JS, 인코딩 등 브라우저에 필요한 정보 제공


 
 
<body> 실제 화면에 표시되는 콘텐츠 영역

사용자에게 보여줄 모든 요소를 담음




<h1> ~ <h6> 제목(Heading) 태그

콘텐츠의 구조와 계층을 표현, SEO에도 중요




<p> 문단(Paragraph) 태그

텍스트를 문단 단위로 구분




<a> 링크(anchor) 태그

다른 페이지나 외부 사이트로 이동할 수 있도록 사용




<img> 이미지 삽입 태그

시각적 콘텐츠를 보여주기 위해 사용




<div> 블록 컨테이너 태그

구역을 나누고 레이아웃을 구성하기 위해 (의미 없음)




<span> 인라인 컨테이너 태그

텍스트 일부에 스타일 적용할 때 사용




<ul> / <ol> / <li> 목록 태그

정보를 나열할 때 (순서 없음/있음), 항목 정의




<table> / <tr> / <td> / <th> 표 태그

데이터를 행과 열로 정리하기 위해 사용




<form> / <input> / <button> 입력 폼 관련 태그

사용자 입력을 서버로 전달하거나 버튼 클릭 이벤트 처리




<header> / <footer> / <section> / <article> / <nav> 시맨틱 태그 (HTML5)

태그 자체가 콘텐츠의 의미와 용도를 설명하는 HTML 요소.




=======================================================================================

✨ 요약
구조 태그: <html>, <head>, <body>

텍스트 태그: <h1>, <p>, <span>

링크/이미지: <a>, <img>

레이아웃 컨테이너: <div>, <section>, <article>

목록/표: <ul>, <ol>, <li>, <table>

폼 입력: <form>, <input>, <button>

시맨틱 태그: <header>, <footer>, <nav>
