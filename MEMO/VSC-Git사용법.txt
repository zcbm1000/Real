git commit -am 
add와 commit 을 같이해줌
단 새로 만든 파일에 한해서는 불가능함.

git add .; git commit -m "테스트 "로
git add . 하고 git commit -m "테스트" 한다.

git add .; git commit -m "테스트 "; git push origin main
git add . 하고 git commit -m "테스트" 하고, git push 한다.

가능은 하나 비 추천함..
