ctrl + /  <!-- text --> 주석 처리와 동일함





html  는 <html> 태그는 웹 문서의 뜻을 나타낸다

meta  는 <meta> 태그는 웹 문서가 출력될 때 사용되는 인코딩을 설정함.
                대부분의 모델에서는 UTF-8 을 사용함
		 
title 는 <title>태그는 웹 문서의 제목을 출력함

head  는 <head> 태그는 웹 문서의 설정 정보를 표기함

body  는 <body> 태그는 웹 문서에서 화면에 출력(랜더링) 되는 정보를 코딩한다

브라우저에서 실행함. 인터프리터통해서



인라인 타입와 블럭타입

인라인 타입은 컨텐츠가 차지하는 공간 만을 뜻함
블럭   타입은  행 전체를 뜻함 (<h>, <p>)






html 태그 목록
<h>           head 의 약자로 머릿말을 뜻함
<p>           paragraph 의 약자로 문단을 뜻 함
<br>          break 의 약자 줄을 바꿀때 사용 함
<strong>      글자의 굵기를 두껍게 표시해줌
<b>           글자의 굵기를 두껍게 표시해줌
<em>          글자를 기울임
<i>           글자를 기울임
<small>       글자를 작게만듬
<mark>        글자에 형광펜 효과를 추가 함
<del>         글자를 삭선 표시 함
<sub>         글자를 아래 첨자 함
<sup>         글자를 윗 첨자 함
&nbsp;        공백을 추가함 
<img>         이미지를 불러옴 src를 이용해야함
<alt>         이미지 불러오기를 실패했을때 띄울 메시지
<ul>          unloader list 의 약자 순서가 없는 목록
<ol>          순서가 있는 목록
<form>        양식을 뜻함
<input>       입력 값 창        
<select>      여러개의 선택지 중 하나를 선택함(option 을 이용하여 선택지 생성)


type = radio 두가지 중 양자 택일
type = email 이메일 입력시 @ 없으면 안됨
